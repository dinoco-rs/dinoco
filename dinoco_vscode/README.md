
# Dinoco for VS Code

Dinoco adds language support for `.dinoco` schema files in Visual Studio Code.

## Features

- Syntax highlighting for Dinoco schemas
- Diagnostics powered by the Dinoco language server
- Document formatting
- Context-aware completions for common Dinoco schema constructs

Source code and issue tracker:
