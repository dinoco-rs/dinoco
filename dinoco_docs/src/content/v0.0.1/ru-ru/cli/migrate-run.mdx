# Как это использовать?

Команда `dinoco migrate run` выполняет все ожидающие миграции.

Она используется для синхронизации настроенной базы данных с уже сгенерированной историей миграций.

---

## Что делает команда

При выполнении этой команды CLI:

- Читает историю миграций
- Проверяет, какие еще не были применены
- Выполняет ожидающие миграции по порядку

## Когда использовать

Используйте эту команду, если:

- Вы уже сгенерировали миграции
- Вы хотите применить ожидающие изменения в базе данных
- Вы готовите локальную среду или среду развертывания

## Следующие шаги

При необходимости сгенерируйте модели с помощью:

```bash
dinoco models generate
```
