# Как использовать?

Команда `dinoco migrate run` выполняет все отложенные миграции.

Используется для выравнивания настроенной базы данных с уже сгенерированной историей миграций.

---

## Что делает команда

При выполнении этой команды CLI:

- Читает историю миграций
- Проверяет, какие еще не были применены
- Выполняет отложенные миграции по порядку

## Когда использовать

Используйте эту команду, когда:

- У вас уже есть сгенерированные миграции
- Вы хотите применить отложенные изменения в базе данных
- Вы готовите локальную среду или среду развертывания

## Следующие шаги

При необходимости сгенерируйте модели с помощью:

```bash
dinoco models generate
```
