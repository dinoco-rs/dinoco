# Как использовать?

Команда `dinoco schema restore` восстанавливает файл `schema.dinoco` из локальной миграции.

Это полезно, если вам нужно воссоздать схему из сохраненной истории.

---

## Что делает команда

Эта команда:

- Читает локальную миграцию
- Получает сохраненные метаданные этой миграции
- Воссоздает файл `schema.dinoco`

## Пример восстановления из определенной миграции

```bash
dinoco schema restore 20250410_create_users
```

## Когда использовать

Используйте эту команду, если:

- Файл `schema.dinoco` был утерян
- Вы хотите воссоздать локальную схему из истории
- Вам нужно сравнить схему, сохраненную в определенной миграции

## Следующие шаги

После восстановления вы можете проверить схему, а затем сгенерировать или применить новые миграции.
