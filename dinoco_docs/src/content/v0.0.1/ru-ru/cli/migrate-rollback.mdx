export const title = "Откат миграции";

# Как использовать?

Команда `dinoco migrate rollback` отменяет последнюю примененную миграцию.

Это полезно, если вам нужно отменить последнее изменение в базе данных.

---

## Что выполняет команда

При выполнении этой команды CLI пытается:

- Идентифицировать последнюю примененную миграцию
- Отменить эту миграцию
- Обновить историю миграций базы данных

## Параметры

[NAME]: Имя миграции (для возврата к определенной) (Необязательно)

## Пример использования

```bash
dinoco migrate rollback
```

## Когда использовать

Используйте эту команду, если:

- Необходимо отменить последнюю миграцию
- Вы обнаружили недавнюю проблему в схеме
- Вы корректируете процесс эволюции базы данных в процессе разработки

## Предосторожности

- Не каждое структурное изменение можно отменить без последствий.
- Проверьте последствия миграции, прежде чем выполнять откат в важных средах.

## Следующие шаги

После того как вы скорректировали схему, вы можете сгенерировать новую миграцию:

```bash
dinoco migrate generate
```
