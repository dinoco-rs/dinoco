export const title = "Откат миграции";

# Как использовать?

Команда `dinoco migrate rollback` отменяет последнюю примененную миграцию.

Это полезно, когда вам нужно отменить последнее изменение в базе данных.

---

## Что делает команда

При выполнении этой команды CLI пытается:

- Идентифицировать последнюю примененную миграцию
- Отменить эту миграцию
- Обновить историю миграций базы данных

## Параметры

[ИМЯ]: Имя миграции (для отката к конкретной) (Необязательно)

## Пример использования

```bash
dinoco migrate rollback
```

## Когда использовать

Используйте эту команду, когда:

- Последняя миграция должна быть отменена
- Вы обнаружили недавнюю проблему в схеме
- Вы настраиваете процесс эволюции базы данных в разработке

## Меры предосторожности

- Не каждое структурное изменение легко отменить без последствий.
- Просмотрите эффект миграции перед выполнением отката в важных средах.

## Следующие шаги

После настройки схемы вы можете сгенерировать новую миграцию:

```bash
dinoco migrate generate
```
