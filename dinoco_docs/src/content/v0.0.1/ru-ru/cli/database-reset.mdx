# Как использовать?

Команда `dinoco database reset` сбрасывает сконфигурированную базу данных.

Это полезно при разработке, когда вы хотите полностью очистить текущее состояние базы данных.

---

## Что делает команда

При выполнении этой команды CLI пытается удалить объекты из текущей базы данных и воссоздать чистое состояние для сконфигурированной среды.

Этот процесс особенно полезен для:

- Перезапуска локальной среды с нуля
- Очистки тестовых данных
- Исправления несогласованных сред во время разработки

## Предосторожности

- Эта команда является разрушительной для сконфигурированной базы данных.
- Используйте с осторожностью вне локальной среды.
- Всегда подтверждайте, что URL-адрес подключения указывает на правильную базу данных.

## Следующие шаги

После сброса вы обычно выполняете следующее:

```bash
dinoco migrate run
```

или:

```bash
dinoco migrate generate --apply
```
