# Как это использовать?

Команда `dinoco migrate generate` генерирует миграцию из текущей схемы.

Она сравнивает текущее состояние схемы с известной историей и создает необходимые артефакты для развития базы данных.

---

## Что делает команда

Эта команда:

- Читает текущую схему
- Генерирует новую локальную миграцию
- Подготавливает артефакты, используемые Dinoco для развития базы данных

При желании она также может немедленно применить миграцию и сгенерировать модели Rust.

## Параметры

### --apply

Немедленно применяет сгенерированную миграцию, а также генерирует модели Rust.

Пример:

```bash
dinoco migrate generate --apply
```

## Пример использования без немедленного применения

```bash
dinoco migrate generate
```

Этот рабочий процесс полезен, если вы хотите:

- Проверить миграцию перед применением
- Проверить изменения в системе контроля версий
- Разделить генерацию и выполнение на разные шаги

## Пример использования с немедленным применением

```bash
dinoco migrate generate --apply
```

Этот рабочий процесс полезен, если вы хотите:

- Быстро обновить локальную базу данных
- Сгенерировать модели сразу после миграции
- Быстрее итерировать во время разработки

## Следующие шаги

После генерации вы можете:

```bash
dinoco migrate run
```

или:

```bash
dinoco models generate
```
