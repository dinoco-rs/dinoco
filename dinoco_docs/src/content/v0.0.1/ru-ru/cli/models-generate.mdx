# Как это использовать?

Команда `dinoco models generate` генерирует Rust-модели из последней миграции, сохраненной в базе данных.

Она используется для согласования сгенерированного кода с уже примененной структурой схемы.

---

## Что делает команда

Эта команда:

- Читает последнюю доступную миграцию
- Реконструирует сгенерированные Rust-модели
- Обновляет артефакты, используемые типизированным Dinoco API

## Когда использовать

Используйте эту команду, если:

- Вы уже применили миграции и хотите перегенерировать код
- Вам нужно обновить модели после структурных изменений
- Вы хотите убедиться, что типизированный клиент отражает последнее состояние

## Следующие шаги

После генерации вы можете использовать модели в своем Rust-коде.
