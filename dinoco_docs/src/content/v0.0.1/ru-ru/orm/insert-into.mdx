export const title = "insert_into";

# insert_into

Используется для вставки записи.

---

## Что можно сделать

- Передать значения с помощью `.values(item)`
- Вставить с отношением через `.with_relation(related)`
- Подключить существующее отношение через `.with_connection(connected)`
- Выполнить с помощью `.execute(&client)`

## Описание методов

- `.values(item)`: определяет вставляемую запись.
- `.with_relation(related)`: вставляет новую, связанную запись вместе с родительской записью.
- `.with_connection(connected)`: связывает вставленную запись с уже существующим элементом, обычно в поддерживаемых потоках отношений для соединения.
- `.returning::<T>()`: изменяет возвращаемое значение на типизированную проекцию вставленного элемента.
- `.execute(&client)`: выполняет операцию записи в базу данных.

## Возвращаемое значение

Без `.returning::<T>()`, возвращаемое значение:

```rust
DinocoResult<()>
```

С `.returning::<T>()`, возвращаемое значение становится:

```rust
DinocoResult<T>
```

## Простой пример

```rust
dinoco::insert_into::<User>()
    .values(User {
        id: "usr_1".to_string(),
        email: "ana@acme.com".to_string(),
        name: "Ana".to_string(),
    })
    .execute(&client)
    .await?;
```

## Пример со связью

Используйте `.with_relation(...)`, если сгенерированная модель поддерживает вставку родительской и связанной записи вместе.

```rust
dinoco::insert_into::<User>()
    .values(user)
    .with_relation(profile)
    .execute(&client)
    .await?;
```

## Пример с подключением

Используйте `.with_connection(...)`, если вы хотите вставить элемент и связать уже существующее отношение.

```rust
dinoco::insert_into::<User>()
    .values(new_user)
    .with_connection(existing_team)
    .execute(&client)
    .await?;
```

## Пример с типизированным результатом

```rust
#[derive(Debug, Clone, dinoco::Extend)]
#[extend(User)]
struct UserSummary {
    id: i64,
    name: String,
}

let created = dinoco::insert_into::<User>()
    .values(User { id: 1, name: "Matheus".to_string() })
    .returning::<UserSummary>()
    .execute(&client)
    .await?;
```
