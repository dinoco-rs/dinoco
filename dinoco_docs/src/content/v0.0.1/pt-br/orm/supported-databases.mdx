# Visão geral

O **Dinoco** oferece suporte nativo aos bancos de dados **PostgreSQL**, **MySQL** e **SQLite**.  
Estamos trabalhando para expandir esse suporte para novos bancos em breve!

> ⚠️
> Atualmente, o Dinoco suporta exclusivamente **URLs de conexão direta**.

---

## Versões suportadas

| Banco de dados | Versão mínima |
| :------------- | :------------ |
| **SQLite**     | `>= 3.25`     |
| **PostgreSQL** | `>= 10`       |
| **MySQL**      | `>= 8.0`      |

---

## Configuração de conexão

A configuração é feita dentro do bloco `config` no seu arquivo de schema.

### SQLite

Ao utilizar SQLite, o arquivo do banco será gerado automaticamente dentro da sua pasta `dinoco`.

```dinoco
config {
  database = "sqlite"
  database_url = "file:./database.sqlite" # env("DATABASE_URL")
}
```

## PostgreSQL ou MySQL

Para bancos externos, insira a string de conexão correspondente:

```dinoco
config {
  database = "postgresql" # ou "mysql"
  database_url = env("DATABASE_URL")
}
```

## Mapeamento de tipos

| Dinoco       | Sqlite   | PostgreSQL       | MySQL            |
| :----------- | :------- | :--------------- | :--------------- |
| **String**   | TEXT     | TEXT             | VARCHAR(255)     |
| **Boolean**  | BOOLEAN  | BOOLEAN          | TINYINT(1)       |
| **Integer**  | INTEGER  | BIGINT           | BIGINT           |
| **Float**    | REAL     | DOUBLE PRECISION | DOUBLE PRECISION |
| **DateTime** | DATETIME | TIMESTAMP        | TIMESTAMP        |
| **Date**     | DATE     | DATE             | DATE             |
| **Json**     | BLOB     | JSONB            | JSON             |
