# 如何使用？

`dinoco migrate run` 命令执行所有待处理的迁移。

它用于使配置的数据库与已生成的迁移历史记录保持一致。

---

## 此命令的作用

执行此命令时，CLI 会：

- 读取迁移历史记录
- 检查哪些尚未应用
- 按顺序执行待处理的迁移

## 何时使用

在以下情况下使用此命令：

- 您已经生成了迁移
- 您想将待处理的更改应用到数据库
- 您正在准备本地或部署环境

## 后续步骤

如有需要，请使用以下命令生成模型：

```bash
dinoco models generate
```
