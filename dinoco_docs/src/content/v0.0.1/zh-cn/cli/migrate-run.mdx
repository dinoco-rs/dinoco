# 如何使用？

`dinoco migrate run` 命令会执行所有待处理的 migration。

它用于让已配置数据库与已经生成的 migration 历史保持一致。

---

## 命令的作用

执行这个命令时，CLI 会：

- 读取迁移历史
- 检查哪些尚未应用
- 以正确的顺序执行待处理的迁移

## 何时使用

当以下情况发生时使用这个命令：

- 您已经生成了迁移
- 你希望把待处理的更改应用到数据库
- 您正在准备本地或部署环境
