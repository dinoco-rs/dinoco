export const title = "回滚迁移";

# 如何使用？

`dinoco migrate rollback` 命令会撤销最近一次已应用的 migration。

当你需要撤销数据库中的最近更改时，它会很有用。

---

## 命令的作用

执行这个命令时，CLI 会尝试：

- 识别最近一次应用的迁移
- 撤销此迁移
- 更新数据库中的 migration 历史

## 参数

[名称]: 迁移的名称（用于回滚到特定迁移）（可选）

## 使用示例

```bash
dinoco migrate rollback
```

## 何时使用

当以下情况发生时使用这个命令：

- 需要撤销最近一次迁移
- 您在模式中发现了最近的问题
- 调整开发中的数据库演进流程

## 注意事项

- 并非所有结构性更改都可以轻松且无影响地撤销。
- 在重要环境中执行回滚之前，请检查迁移的影响。

## 下一步

调整模式后，您可以生成一个新的迁移：

```bash
dinoco migrate generate
```
