# 如何使用？

`dinoco models generate` 命令会根据数据库中保存的最后一次 migration 生成 Rust 模型。

它用于使生成的代码与已应用的模式结构保持一致。

---

## 命令的作用

此命令会：

- 读取最后一次可用的迁移
- 重构生成的 Rust 模型
- 更新类型化的 Dinoco API 所使用的工件

## 何时使用

当以下情况发生时使用这个命令：

- 您已经应用了迁移并希望重新生成代码
- 您需要在结构更改后更新模型
- 您希望确保类型化客户端反映最新状态

## 下一步

生成后，您现在可以在 Rust 代码中使用这些模型。
