# 概览

**Dinoco** 原生支持 **PostgreSQL**、**MySQL** 和 **SQLite**。  
我们正在努力尽快把这项支持扩展到更多数据库。

> ⚠️
> 目前 Dinoco 仅支持 **直接连接 URL**。

---

## 支持的版本

| 数据库 | 最低版本 |
| :------------- | :------------ |
| **SQLite**     | `>= 3.25`     |
| **PostgreSQL** | `>= 10`       |
| **MySQL**      | `>= 8.0`      |

---

## 连接配置

配置在 schema 文件中的 `config` 块里完成。

### SQLite

使用 SQLite 时，数据库文件会自动生成在 `dinoco` 文件夹中。

```dinoco
config {
  database = "sqlite"
  database_url = "file:./database.sqlite" # env("DATABASE_URL")
}
```

## PostgreSQL 或 MySQL

对于外部数据库，请填写对应的连接字符串：

```dinoco
config {
  database = "postgresql" # 或 "mysql"
  database_url = env("DATABASE_URL")
}
```

## 类型映射

| Dinoco       | Sqlite   | PostgreSQL       | MySQL            |
| :----------- | :------- | :--------------- | :--------------- |
| **String**   | TEXT     | TEXT             | VARCHAR(255)     |
| **Boolean**  | BOOLEAN  | BOOLEAN          | TINYINT(1)       |
| **Integer**  | INTEGER  | BIGINT           | BIGINT           |
| **Float**    | REAL     | DOUBLE PRECISION | DOUBLE PRECISION |
| **DateTime** | DATETIME | TIMESTAMP        | TIMESTAMP        |
| **Date**     | DATE     | DATE             | DATE             |
| **Json**     | BLOB     | JSONB            | JSON             |
