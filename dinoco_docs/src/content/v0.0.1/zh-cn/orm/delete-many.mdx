export const title = "delete_many";

# delete_many

用于通过一个或多个条件删除多个记录。

---

## 你可以做什么

- `.cond(...)`: 添加用于批量删除的过滤器。
- `.execute(&client)`: 执行删除相应的记录。

## 返回值

`delete_many` 的返回值是：

```rust
DinocoResult<()>
```

## 基础示例

这只会删除满足条件的记录。

```rust
dinoco::delete_many::<Session>()
    .cond(|w| w.expiresAt.lt(dinoco::Utc::now()))
    .execute(&client)
    .await?;
```

## 无过滤器示例

这会删除表中所有数据。

```rust
dinoco::delete_many::<TaskAssignees>()
    .execute(&client)
    .await?;
```
