# Dinoco 简介

这个库的创建旨在简化数据库管理，灵感来源于 [Prisma](https://www.prisma.io/) 的理念，Dinoco 为 [Rust](https://www.rust-lang.org/) 生态系统带来了流畅、类型化且极其高效的接口。

---

## Rust 中的 Prisma 体验

Dinoco 的诞生源于将 Prisma 的敏捷性和强类型带入 [Rust](https://www.rust-lang.org/) 生态系统的愿望。我们相信您不应该花费数小时配置复杂的宏或为日常操作手动编写 SQL。

## 项目支柱

- **开发者体验 (DX) 至上：** 流畅直观的 API，减少认知负担，加速开发。
- **端到端类型安全：** 我们确保您的数据始终与类型定义同步，并在编译时捕获错误。
- **简化关系：** 告别繁琐的手动连接。定义您的关系，让 Dinoco 在幕后解决复杂性。
- **强大的 CLI：** 集成的迁移管理和代码生成，让您的工作流程保持有序和快速。

## 我们的定位

[Rust](https://www.rust-lang.org/) 生态系统拥有强大的工具，但它们往往需要陡峭的学习曲线。**Dinoco** 的出现旨在填补现有选项之间的空白：

- **对比 [Diesel](https://diesel.rs/)：** 尽管 Diesel 需要深入的 SQL 知识并拥有复杂的 DSL（领域特定语言），但 Dinoco 优先考虑简洁且注重生产力的语法，您无需为简单的查询与编译器作斗争。
- **对比 [SeaORM](https://www.sea-ql.org/SeaORM/)：** SeaORM 非常出色，但其基于分层的架构对于许多项目来说可能过于冗长。Dinoco 消除了不必要的“样板代码”，以更少的代码行提供相同的健壮性。

如果您正在寻找 [Rust](https://www.rust-lang.org/) 的性能以及 Prisma 为 Node.js 世界带来的流畅性，那么 Dinoco 是您下一个项目的正确选择。
