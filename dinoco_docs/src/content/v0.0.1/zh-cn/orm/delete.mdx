export const title = "delete";

# delete

用于使用显式过滤器进行删除。

---

## 你可以做什么

- `.cond(...)`: 定义要删除的记录。
- `.execute(&client)`: 在数据库中执行删除操作。

## 返回值

delete 的返回值是：

```rust
DinocoResult<()>
```

## 基础示例

```rust
dinoco::delete::<User>()
    .cond(|w| w.id.eq(10))
    .execute(&client)
    .await?;
```

## 另一个过滤器示例

```rust
dinoco::delete::<Session>()
    .cond(|w| w.token.eq("session-1"))
    .execute(&client)
    .await?;
```
