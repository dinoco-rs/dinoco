# Overview

Dinoco Client provides a query layer focused on productivity, typing, and operational predictability.

## Installation

- Add the client package to your project.
- Generate artifacts from the schema.
- Configure the database connection.

## First query

```ts
import { createClient } from '@dinoco/client';

const db = createClient();

const users = await db.user.findMany({
	limit: 10,
});
```
