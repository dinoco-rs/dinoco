# How to use?

The command `dinoco models generate` generates the Rust models from the latest migration stored in the database.

It is used to keep the generated code aligned with the already applied schema structure.

---

## What the command does

This command:

- Reads the latest available migration
- Reconstructs the generated Rust models
- Updates the artifacts used by Dinoco's typed API

## When to use

Use this command when:

- You have already applied migrations and want to regenerate the code
- You need to update the models after structural changes
- You want to ensure that the typed client reflects the latest state

## Next steps

After generation, you can now use the models in your Rust code.
