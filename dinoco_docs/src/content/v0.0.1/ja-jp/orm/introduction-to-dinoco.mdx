# Dinoco の紹介

このライブラリは、データベース管理をシンプルにするために作られました。[Prisma](https://www.prisma.io/) の思想に着想を得て、Dinoco は流れるようで型安全かつ高い生産性を持つインターフェースを [Rust](https://www.rust-lang.org/) エコシステムにもたらします。

---

## Rust における Prisma の体験

Dinoco は、Prisma の俊敏さと強い型付けを [Rust](https://www.rust-lang.org/) エコシステムに持ち込みたいという思いから生まれました。日常的な操作のために、複雑なマクロの設定や手書き SQL に何時間も費やす必要はないと私たちは考えています。

## プロジェクトの柱

- **DX（開発者体験）を最優先:** 認知負荷を軽減し、開発を加速させる、流れるような直感的なAPI。
- **エンドツーエンドの型安全性:** データが常に型定義と同期していることを保証し、コンパイル時にエラーを捕捉します。
- **シンプルなリレーションシップ:** 面倒な手動結合は忘れてください。リレーションシップを定義すれば、Dinocoがバックグラウンドで複雑さを解決します。
- **強力なCLI:** ワークフローを整理し、迅速に保つための統合されたマイグレーション管理とコード生成。

## 私たちはどこに位置するのか？

[Rust](https://www.rust-lang.org/)エコシステムには強力なツールがありますが、多くの場合、急な学習曲線が必要です。**Dinoco**は、現在の選択肢間のギャップを埋めます。

- **[Diesel](https://diesel.rs/)との比較:** Dieselが深いSQL知識と複雑なDSL（ドメイン固有言語）を要求するのに対し、Dinocoは、簡単なクエリのためにコンパイラと格闘することなく、クリーンで生産性重視の構文を優先します。
- **[SeaORM](https://www.sea-ql.org/SeaORM/)との比較:** SeaORMは優れていますが、そのレイヤーベースのアーキテクチャは多くのプロジェクトにとって過度に冗長になる可能性があります。Dinocoは不要な「ボイラープレート」コードを排除し、はるかに少ないコード行で同じ堅牢性を提供します。

もしあなたが[Rust](https://www.rust-lang.org/)のパフォーマンスと、PrismaがNode.jsの世界にもたらした流動性を求めているなら、Dinocoはあなたの次のプロジェクトに最適な選択です。
