# 使い方

`dinoco schema restore` コマンドは、ローカル migration から `schema.dinoco` ファイルを復元します。

保存された履歴からスキーマを再構築する必要がある場合に便利です。

---

## コマンドの動作

このコマンドは：

- ローカルマイグレーションを読み取ります
- このマイグレーションの保存されたメタデータを取得します
- `schema.dinoco` ファイルを再作成します

## 特定のマイグレーションから復元する例

```bash
dinoco schema restore 20250410_create_users
```

## 使うタイミング

次のようなときにこのコマンドを使います:

- `schema.dinoco` ファイルが失われた場合
- 履歴からローカルスキーマを再構築したい場合
- 特定のマイグレーションで保存されたスキーマを比較する必要がある場合

## 次のステップ

復元後、スキーマを確認し、新しいマイグレーションを生成または適用できます。
