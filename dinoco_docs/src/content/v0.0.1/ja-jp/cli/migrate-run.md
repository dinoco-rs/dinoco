# 使い方

`dinoco migrate run` コマンドは、保留中のすべてのマイグレーションを実行します。

これは、設定されたデータベースを、すでに生成されたマイグレーション履歴と同期させるために使用されます。

---

## コマンドの機能

このコマンドを実行すると、CLI は次のことを行います。

- マイグレーション履歴を読み込みます
- まだ適用されていないものを確認します
- 保留中のマイグレーションを順番に実行します

## 使用するタイミング

このコマンドは次の場合に使用します。

- すでにマイグレーションが生成されている場合
- データベースに保留中の変更を適用したい場合
- ローカル環境またはデプロイ環境を準備している場合

## 次のステップ

必要に応じて、次のコマンドでモデルを生成します。

```bash
dinoco models generate
```
