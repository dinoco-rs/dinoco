# 使用方法

`dinoco database reset` コマンドは、設定されたデータベースをリセットします。

開発中にデータベースの現在の状態を完全にクリアしたい場合に便利です。

---

## コマンドの機能

このコマンドを実行すると、CLI は現在のデータベースからオブジェクトを削除し、設定された環境のためにクリーンな状態を再作成しようとします。

このフローは特に以下の状況で役立ちます。

- ローカル環境をゼロから再起動する
- テストデータをクリアする
- 開発中に一貫性のない環境を修正する

## 注意事項

- このコマンドは、設定されたデータベースに対して破壊的です。
- ローカル環境以外では注意して使用してください。
- 接続 URL が正しいデータベースを指していることを常に確認してください。

## 次のステップ

リセット後、通常は以下を実行します。

```bash
dinoco migrate run
```

または：

```bash
dinoco migrate generate --apply
```
