# 使い方

`dinoco migrate run` コマンドは、保留中の migration をすべて実行します。

設定されたデータベースを、すでに生成済みの migration 履歴と同期するために使われます。

---

## コマンドの動作

このコマンドを実行すると、CLI は次の処理を行います:

- マイグレーション履歴を読み取ります
- まだ適用されていないものを確認します
- 保留中のマイグレーションを正しい順序で実行します

## 使うタイミング

次のようなときにこのコマンドを使います:

- すでにマイグレーションを生成している場合
- データベースに保留中の変更を適用したい
- ローカル環境またはデプロイ環境を準備する場合

## 次のステップ
