export const title = "insert_into";

# insert_into

Utilisé pour insérer un enregistrement.

---

## Ce que vous pouvez faire

- Passer des valeurs avec `.values(item)`
- Insérer avec une relation via `.with_relation(related)`
- Connecter une relation existante via `.with_connection(connected)`
- Exécuter avec `.execute(&client)`

## Description des méthodes

- `.values(item)`: définit l'enregistrement à insérer.
- `.with_relation(related)`: insère un nouvel enregistrement lié avec l'élément parent.
- `.with_connection(connected)`: connecte l'enregistrement inséré à un élément déjà existant, généralement dans des flux de relations qui supportent la connexion.
- `.returning::<T>()`: modifie le retour en une projection typée de l'élément inséré.
- `.execute(&client)`: exécute l'opération d'écriture dans la base de données.

## Retour

Sans `.returning::<T>()`, le retour est :

```rust
DinocoResult<()>
```

Avec `.returning::<T>()`, le retour devient :

```rust
DinocoResult<T>
```

## Exemple simple

```rust
dinoco::insert_into::<User>()
    .values(User {
        id: "usr_1".to_string(),
        email: "ana@acme.com".to_string(),
        name: "Ana".to_string(),
    })
    .execute(&client)
    .await?;
```

## Exemple avec relation

Utilisez `.with_relation(...)` lorsque le modèle généré supporte l'insertion conjointe de l'élément parent et de l'élément lié.

```rust
dinoco::insert_into::<User>()
    .values(user)
    .with_relation(profile)
    .execute(&client)
    .await?;
```

## Exemple avec connexion

Utilisez `.with_connection(...)` si vous souhaitez insérer un élément et connecter une relation déjà existante.

```rust
dinoco::insert_into::<User>()
    .values(new_user)
    .with_connection(existing_team)
    .execute(&client)
    .await?;
```

## Exemple avec retour typé

```rust
#[derive(Debug, Clone, dinoco::Extend)]
#[extend(User)]
struct UserSummary {
    id: i64,
    name: String,
}

let created = dinoco::insert_into::<User>()
    .values(User { id: 1, name: "Matheus".to_string() })
    .returning::<UserSummary>()
    .execute(&client)
    .await?;
```
