# delete_many

Est utilisé pour supprimer plusieurs entrées avec une ou plusieurs conditions.

---

## Ce que vous pouvez faire

- `.cond(...)`: ajoute des filtres pour la suppression en masse.
- `.execute(&client)`: exécute la suppression des entrées correspondantes.

## Retour

Le retour de `delete_many` est :

```rust
DinocoResult<()>
```

## Exemple de base

Ceci supprime uniquement les entrées qui correspondent à la condition.

```rust
dinoco::delete_many::<Session>()
    .cond(|w| w.expiresAt.lt(dinoco::Utc::now()))
    .execute(&client)
    .await?;
```

## Exemple sans filtre

Ceci supprime toutes les données de la table.

```rust
dinoco::delete_many::<TaskAssignees>()
    .execute(&client)
    .await?;
```
