# Installation

Installez la CLI pour démarrer des projets, générer des artefacts et valider l'environnement local.

## Installation

```bash
cargo install dinoco
```

## Configuration

- Personnalisez les variables d'environnement.
- Définissez le fournisseur de base de données.
- Générez les fichiers d'espace de travail initiaux.
