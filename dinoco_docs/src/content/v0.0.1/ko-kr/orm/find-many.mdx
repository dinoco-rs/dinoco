# find_many

레코드 목록을 검색하는 데 사용됩니다.

---

## 할 수 있는 것

- `.select::<T>()`를 사용하여 프로젝션 선택
- `.cond(...)`를 사용하여 필터링
- `.take(...)`를 사용하여 제한
- `.skip(...)`를 사용하여 페이지 매김
- `.order_by(...)`를 사용하여 정렬
- `.includes(...)`를 사용하여 관계 로드
- `.count(...)`를 사용하여 관계 개수 세기
- `.read_in_primary()`를 사용하여 기본 데이터베이스에서 읽기 강제
- `.execute(&client)`를 사용하여 실행

## 메서드 설명

- `.select::<T>()`: 모델의 기본 프로젝션을 사용자 정의 프로젝션으로 대체합니다.
- `.cond(...)`: 쿼리에 필터 조건을 추가합니다.
- `.take(...)`: 반환되는 레코드 수를 제한합니다.
- `.skip(...)`: 결과가 반환되기 전에 지정된 수의 레코드를 건너뜠니다.
- `.order_by(...)`: 쿼리의 정렬 순서를 정의합니다.
- `.includes(...)`: 주 레코드와 함께 관계를 로드합니다.
- `.count(...)`: 관계 개수를 계산하고 `posts_count`와 같은 필드를 채웁니다.
- `.read_in_primary()`: 복제본을 사용하지 않고 기본 데이터베이스에서 읽기를 강제합니다.
- `.execute(&client)`: 데이터베이스에서 쿼리를 실행합니다.

## 반환값

`select::<T>()`가 없으면 반환값은 다음과 같습니다:

```rust
DinocoResult<Vec<M>>
```

`select::<T>()`를 사용하면 반환값은 다음과 같이 변경됩니다:

```rust
DinocoResult<Vec<T>>
```

## 기본 예제

```rust
let users = dinoco::find_many::<User>()
    .execute(&client)
    .await?;
```

## 필터 예제

```rust
let users = dinoco::find_many::<User>()
    .cond(|w| w.email.eq("ana@acme.com"))
    .execute(&client)
    .await?;
```

## 페이지네이션과 정렬 예제

```rust
let users = dinoco::find_many::<User>()
    .order_by(|w| w.name.asc())
    .skip(20)
    .take(10)
    .execute(&client)
    .await?;
```

## 사용자 정의 select 예제

```rust
#[derive(Debug, Clone, dinoco::Extend)]
#[extend(User)]
struct UserListItem {
    id: i64,
    name: String,
}

let users = dinoco::find_many::<User>()
    .select::<UserListItem>()
    .execute(&client)
    .await?;
```

## 단순 include 예제

```rust
#[derive(Debug, Clone, dinoco::Extend)]
#[extend(User)]
struct UserWithPosts {
    id: i64,
    name: String,
    posts: Vec<Post>,
}

let users = dinoco::find_many::<User>()
    .select::<UserWithPosts>()
    .includes(|i| i.posts())
    .execute(&client)
    .await?;
```

## 필터링된 include 예제

```rust
#[derive(Debug, Clone, dinoco::Extend)]
#[extend(User)]
struct UserWithPublishedPosts {
    id: i64,
    name: String,
    posts: Vec<Post>,
}

let users = dinoco::find_many::<User>()
    .select::<UserWithPublishedPosts>()
    .includes(|i| i.posts().cond(|w| w.published.eq(true)))
    .execute(&client)
    .await?;
```

## 중첩 include 예제

```rust
#[derive(Debug, Clone, dinoco::Extend)]
#[extend(Comment)]
struct CommentListItem {
    id: i64,
    text: String,
}

#[derive(Debug, Clone, dinoco::Extend)]
#[extend(Post)]
struct PostWithComments {
    id: i64,
    title: String,
    comments: Vec<CommentListItem>,
    comments_count: usize,
}

#[derive(Debug, Clone, dinoco::Extend)]
#[extend(User)]
struct UserWithPosts {
    id: i64,
    name: String,
    posts: Vec<PostWithComments>,
}

let users = dinoco::find_many::<User>()
    .select::<UserWithPosts>()
    .includes(|i| {
        i.posts()
            .includes(|post| post.comments().take(3))
            .count(|post| post.comments())
    })
    .execute(&client)
    .await?;
```

## 관계 수 예제

```rust
#[derive(Debug, Clone, dinoco::Extend)]
#[extend(User)]
struct UserWithPostsCount {
    id: i64,
    name: String,
    posts_count: usize,
}

let users = dinoco::find_many::<User>()
    .select::<UserWithPostsCount>()
    .count(|i| i.posts())
    .execute(&client)
    .await?;
```

## 기본 데이터베이스에서 읽는 예제

```rust
let users = dinoco::find_many::<User>()
    .read_in_primary()
    .take(5)
    .execute(&client)
    .await?;
```
