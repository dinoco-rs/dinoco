# 사용 방법

`dinoco database reset` 명령은 구성된 데이터베이스를 초기화합니다.

개발 환경에서 데이터베이스의 현재 상태를 완전히 정리하고 싶을 때 유용합니다.

---

## 명령이 하는 일

이 명령을 실행하면 CLI는 현재 데이터베이스에서 객체를 제거하고 구성된 환경을 위한 깨끗한 상태를 다시 생성하려고 시도합니다.

이 과정은 다음 경우에 특히 유용합니다:

- 로컬 환경을 처음부터 다시 시작할 때
- 테스트 데이터를 정리할 때
- 개발 중 일관성 없는 환경을 수정할 때

## 주의사항

- 이 명령은 구성된 데이터베이스에 파괴적입니다.
- 로컬 환경 외부에서는 주의해서 사용하십시오.
- 항상 연결 URL이 올바른 데이터베이스를 가리키는지 확인하십시오.

## 다음 단계

초기화 후에는 일반적으로 다음을 실행합니다:

```bash
dinoco migrate run
```

또는:

```bash
dinoco migrate generate --apply
```
