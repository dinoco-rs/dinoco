# 사용 방법

`dinoco models generate` 명령어는 데이터베이스에 저장된 최신 마이그레이션으로부터 Rust 모델을 생성합니다.

이 명령어는 이미 적용된 스키마 구조와 생성된 코드를 일치시키는 데 사용됩니다.

---

## 명령이 하는 일

이 명령어는 다음을 수행합니다:

- 사용 가능한 최신 마이그레이션을 읽습니다.
- 생성된 Rust 모델을 재구성합니다.
- 타입이 지정된 Dinoco API에서 사용되는 아티팩트를 업데이트합니다.

## 사용 시점

다음과 같은 경우에 이 명령어를 사용하십시오:

- 이미 마이그레이션을 적용했고 코드를 다시 생성하려는 경우
- 구조적 변경 후 모델을 업데이트해야 하는 경우
- 타입이 지정된 클라이언트가 최신 상태를 반영하는지 확인하려는 경우
