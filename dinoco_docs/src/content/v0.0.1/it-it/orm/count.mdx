export const title = "count";

# count

Viene utilizzato per contare i record.

---

## Cosa puoi fare

- Filtrare con `.cond(...)`
- Eseguire con `.execute(&client)`

## Descrizione dei metodi

- `.cond(...)`: limita quali record vengono inclusi nel conteggio.
- `.execute(&client)`: esegue il conteggio nel database.

## Ritorno

Il ritorno di `count` è:

```rust
DinocoResult<usize>
```

## Esempio base

```rust
let total = dinoco::count::<User>()
    .execute(&client)
    .await?;
```

## Esempio con filtro booleano

```rust
let total = dinoco::count::<User>()
    .cond(|w| w.active.eq(true))
    .execute(&client)
    .await?;
```

## Esempio con filtro testuale

```rust
let total = dinoco::count::<User>()
    .cond(|w| w.name.includes("Ana"))
    .execute(&client)
    .await?;
```
