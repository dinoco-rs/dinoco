export const title = "delete";

# delete

Utilizzato per eliminare con un filtro esplicito.

---

## Cosa puoi fare

- `.cond(...)`: definisce quale record deve essere rimosso.
- `.execute(&client)`: esegue la rimozione nel database.

## Ritorno

Il valore di ritorno di `delete` è:

```rust
DinocoResult<()>
```

## Esempio base

```rust
dinoco::delete::<User>()
    .cond(|w| w.id.eq(10))
    .execute(&client)
    .await?;
```

## Esempio con un altro filtro

```rust
dinoco::delete::<Session>()
    .cond(|w| w.token.eq("session-1"))
    .execute(&client)
    .await?;
```
