# Installation

Installieren Sie die CLI, um Projekte zu starten, Artefakte zu generieren und die lokale Umgebung zu validieren.

## Installation

```bash
cargo install dinoco
```

## Konfiguration

- Passen Sie die Umgebungsvariablen an.
- Definieren Sie den Datenbankanbieter.
- Generieren Sie die initialen Workspace-Dateien.
