# Wie benutzt man es?

Der Befehl `dinoco schema restore` stellt die Datei `schema.dinoco` aus einer lokalen Migration wieder her.

Dies ist nützlich, wenn Sie das Schema aus dem gespeicherten Verlauf neu erstellen müssen.

---

## Was der Befehl tut

Dieser Befehl:

- Liest eine lokale Migration
- Ruft die gespeicherten Metadaten dieser Migration ab
- Erstellt die Datei `schema.dinoco` neu

## Beispiel für die Wiederherstellung aus einer spezifischen Migration

```bash
dinoco schema restore 20250410_create_users
```

## Wann zu verwenden

Verwenden Sie diesen Befehl, wenn:

- Die Datei `schema.dinoco` verloren gegangen ist
- Sie das lokale Schema aus dem Verlauf neu erstellen möchten
- Sie das in einer spezifischen Migration gespeicherte Schema vergleichen müssen

## Nächste Schritte

Nach der Wiederherstellung können Sie das Schema überprüfen und dann neue Migrationen generieren oder anwenden.
