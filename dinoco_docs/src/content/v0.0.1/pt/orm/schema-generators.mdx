# Generators

Generators conectam o schema ao restante da ferramenta, produzindo client, tipos auxiliares e artefatos de integracao.

## Config

Declare os generators no topo do arquivo para deixar claro o que sera produzido a partir do schema.

```dinoco
generator client {
	provider = "dinoco-client-js"
	output   = "./generated/client"
}
```

## Generate

Execute o fluxo de geracao sempre que campos, modelos ou relacoes forem alterados.

## Artefatos

Os artefatos podem incluir client, declaracoes de tipo e arquivos de suporte usados por outras partes da stack.

## Pipeline

Em CI, combine validacao de schema com geracao para evitar divergencia entre o codigo versionado e o modelo real.
