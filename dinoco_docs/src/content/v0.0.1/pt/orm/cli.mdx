# Visao geral

O Dinoco Client entrega uma camada de consultas com foco em produtividade, tipagem e previsibilidade operacional.

## Instalacao

- Adicione o pacote do client ao projeto.
- Gere os artefatos do schema.
- Configure a conexao com o banco.

## Primeira consulta

```ts
import { createClient } from '@dinoco/client';

const db = createClient();

const users = await db.user.findMany({
	limit: 10,
});
```
