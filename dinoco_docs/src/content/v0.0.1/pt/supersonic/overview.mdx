# Arquitetura

Supersonic e o runtime orientado a throughput para workloads de alta concorrencia.

## Workers

- Isole filas pesadas.
- Distribua jobs por dominio.
- Mantenha observabilidade por worker.

## Escalonamento

O runtime foi pensado para crescer horizontalmente sem perder previsibilidade.

## Deploy

Use pipelines separados para runtime, migrations e assets de frontend.
