# Studio

O Studio oferece uma camada visual para inspecionar dados, navegar por tabelas e validar resultados sem sair do fluxo de docs.

## Abrir

Abra o Studio quando quiser conferir dados reais apos gerar o schema ou aplicar uma migration.

```bash
dinoco studio
```

## Explorar

A navegacao por colecoes e relacoes deve ajudar a entender o modelo antes de alterar consultas ou estrutura.

## Filtrar

Filtros rapidos ajudam a localizar registros especificos e revisar mudancas de forma mais segura.

## Colaboracao

Use capturas, exemplos e consultas repetiveis para alinhar conversas entre produto, backend e plataforma.
