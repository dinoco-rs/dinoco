# dinoco init

O comando `dinoco init` cria a estrutura inicial do projeto com a configuracao recomendada para comecar rapido.

## Template

Escolha um template enxuto quando quiser explorar a stack ou um template completo para iniciar um projeto de produto.

```bash
dinoco init my-app
```

## Estrutura

O comando pode criar `schema`, pasta de gerados, arquivo de ambiente e scripts basicos para desenvolvimento.

## Config

Revise os arquivos criados antes do primeiro commit para alinhar caminhos, nomes de pacote e a conexao do banco.

## Proximos passos

Depois do bootstrap, rode a geracao inicial, confira o schema e execute um comando de diagnostico para validar o ambiente.
