# Instalacao

Instale o CLI para iniciar projetos, gerar artefatos e validar o ambiente local.

## Init

```bash
cargo install dinoco
dinoco init
```

## Configuracao

- Ajuste variaveis de ambiente.
- Defina o provider do banco.
- Gere os arquivos iniciais do workspace.

## Verificacao

Use `dinoco doctor` para validar dependencias e setup local.
