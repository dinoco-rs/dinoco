# Introdução ao Dinoco

Essa biblioteca foi criada com o intuito de facilitar o gerenciamento de banco de dados, inspirado pela filosofia do [Prisma](https://www.prisma.io/), o Dinoco traz para o ecossistema [Rust](https://www.rust-lang.org/) uma interface fluida, tipada e extremamente produtiva.

---

## A Experiência Prisma em Rust

O Dinoco nasce da vontade de trazer a agilidade e a tipagem forte do Prisma para o ecossistema [Rust](https://www.rust-lang.org/) . Acreditamos que você não deve perder horas configurando macros complexas ou escrevendo SQL manual para operações comuns do dia a dia.

## Pilares do Projeto

- **DX (Experiência do desenvolvedor) em primeiro lugar:** Uma API fluida e intuitiva que reduz a carga cognitiva e acelera o desenvolvimento.
- **Segurança de tipos ponta a ponta:** Garantimos que seus dados estejam sempre em sincronia com suas definições de tipos, capturando erros ainda em tempo de compilação.
- **Relacionamentos descomplicados:** Esqueça os joins manuais exaustivos. Defina suas relações e deixe que o Dinoco resolva a complexidade por baixo dos panos.
- **CLI poderosa:** Gerenciamento de migrações e geração de código integrados para manter seu fluxo de trabalho organizado e rápido.

## Onde nos posicionamos?

O ecossistema [Rust](https://www.rust-lang.org/) possui ferramentas poderosas, mas que muitas vezes exigem uma curva de aprendizado íngreme. O **Dinoco** surge para preencher o espaço entre as opções atuais:

- **Frente ao [Diesel](https://diesel.rs/):** Enquanto o Diesel exige um conhecimento profundo de SQL e possui uma DSL (Domain Specific Language) complexa, o Dinoco prioriza uma sintaxe limpa e focada em produtividade, sem que você precise lutar contra o compilador para consultas simples.
- **Frente ao [SeaORM](https://www.sea-ql.org/SeaORM/):** O SeaORM é excelente, mas sua arquitetura baseada em camadas pode ser excessivamente verbosa para muitos projetos. O Dinoco remove o "código repetitivo" desnecessário, entregando a mesma robustez com muito menos linhas de código.

Se você busca a performance do [Rust](https://www.rust-lang.org/) com a fluidez que o Prisma trouxe para o mundo Node.js, o Dinoco é a escolha certa para o seu próximo projeto.
