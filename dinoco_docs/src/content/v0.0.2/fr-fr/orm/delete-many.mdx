# delete_many

Utilisé pour supprimer plusieurs enregistrements avec une ou plusieurs conditions.

---

## Ce que vous pouvez faire

- `.cond(...)`: ajoute des filtres pour la suppression par lots.
- `.execute(&client)`: exécute la suppression des enregistrements correspondants.

## Retour

Le retour de `delete_many` est :

```rust
DinocoResult<()>
```

## Exemple de base

Cela supprime uniquement les enregistrements qui remplissent la condition.

```rust
dinoco::delete_many::<Session>()
    .cond(|w| w.expiresAt.lt(dinoco::Utc::now()))
    .execute(&client)
    .await?;
```

## Exemple sans filtre

Cela supprime toutes les données de la table.

```rust
dinoco::delete_many::<TaskAssignees>()
    .execute(&client)
    .await?;
```
