# How to use?

The command `dinoco models generate` generates the Rust models from the last migration stored in the database.

It is used to keep the generated code aligned with the already applied schema structure.

---

## What the command does

This command:

- Reads the last available migration
- Rebuilds the generated Rust models
- Updates the artifacts used by the Dinoco typed API

## When to use

Use this command when:

- You have already applied migrations and want to regenerate the code
- You need to update the models after structural changes
- You want to ensure the typed client reflects the latest state

## Next steps

After generation, you can now use the models in your Rust code.
