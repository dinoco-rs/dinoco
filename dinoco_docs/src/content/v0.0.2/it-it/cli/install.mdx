# Installazione

Installa la CLI per avviare progetti, generare artefatti e convalidare l'ambiente locale.

## Installazione

```bash
cargo install dinoco
```

## Configurazione

- Adatta le variabili d'ambiente.
- Definisci il provider del database.
- Genera i file iniziali dello spazio di lavoro.
