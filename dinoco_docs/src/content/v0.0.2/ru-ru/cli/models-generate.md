# Как использовать?

Команда `dinoco models generate` генерирует модели Rust из последней миграции, хранящейся в базе данных.

Она используется для поддержания сгенерированного кода в соответствии со структурой уже примененной схемы.

---

## Что делает команда

Эта команда:

- Читает последнюю доступную миграцию
- Перестраивает сгенерированные модели Rust
- Обновляет артефакты, используемые типизированным API Dinoco

## Когда использовать

Используйте эту команду, когда:

- Вы уже применили миграции и хотите регенерировать код
- Вам нужно обновить модели после структурных изменений
- Вы хотите убедиться, что типизированный клиент отражает последнее состояние

## Следующие шаги

После генерации вы можете использовать модели в своем коде Rust.
