# Как использовать?

Команда `dinoco schema restore` восстанавливает файл `schema.dinoco` из локальной миграции.

Это полезно, когда вам нужно восстановить схему из сохраненной истории.

---

## Что делает команда

Эта команда:

- Читает локальную миграцию
- Извлекает сохраненные метаданные из этой миграции
- Воссоздает файл `schema.dinoco`

## Пример восстановления из конкретной миграции

```bash
dinoco schema restore 20250410_create_users
```

## Когда использовать

Используйте эту команду, когда:

- Файл `schema.dinoco` был утерян
- Вы хотите восстановить локальную схему из истории
- Вам нужно сравнить схему, сохраненную в конкретной миграции

## Следующие шаги

После восстановления вы можете просмотреть схему, а затем сгенерировать или применить новые миграции.
