# Как использовать?

Команда `dinoco init` инициализирует среду Dinoco в текущем проекте.

Она подготавливает необходимую основу для начала работы со схемой, миграциями и генерацией моделей.

---

## Что делает команда

При выполнении `dinoco init` CLI руководит начальной настройкой среды Dinoco.

В целом, этот процесс подготавливает:

- Базовая структура каталога `dinoco`
- Начальные файлы схемы
- Конфигурация базы данных
- Основа для будущих миграций

## Когда использовать

Используйте эту команду, когда:

- Вы настраиваете Dinoco впервые в проекте
- Вы хотите создать начальную структуру локальной среды
- Вам нужно подготовить проект для генерации миграций и моделей

## Следующие шаги

После `init` наиболее распространенный процесс:

```bash
dinoco migrate generate
```

или, если вы хотите сгенерировать и применить немедленно:

```bash
dinoco migrate generate --apply
```
