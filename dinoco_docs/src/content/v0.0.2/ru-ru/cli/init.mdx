# Как использовать?

Команда `dinoco init` инициализирует окружение Dinoco в текущем проекте.

Она подготавливает необходимую основу для начала работы со схемой, миграциями и генерацией моделей.

---

## Что делает команда

При выполнении `dinoco init` CLI проводит через начальную конфигурацию окружения Dinoco.

В целом, этот процесс подготавливает:

- Базовую структуру каталога `dinoco`
- Начальные файлы схемы
- Конфигурацию базы данных
- Основу для будущих миграций

## Когда использовать

Используйте эту команду, если:

- Вы настраиваете Dinoco в проекте впервые
- Вы хотите создать начальную структуру локального окружения
- Вам нужно подготовить проект для генерации миграций и моделей

## Следующие шаги

После `init` наиболее распространенный процесс:

```bash
dinoco migrate generate
```

или, если вы хотите сразу сгенерировать и применить:

```bash
dinoco migrate generate --apply
```
