# Установка

Установите CLI для запуска проектов, генерации артефактов и проверки локальной среды.

## Установка

```bash
cargo install dinoco
```

## Настройка

- Настройте переменные среды.
- Определите поставщика базы данных.
- Сгенерируйте начальные файлы рабочего пространства.
