# Как использовать?

Команда `dinoco migrate generate` генерирует миграцию на основе текущей схемы.

Она сравнивает текущее состояние схемы с известной историей и создает необходимые артефакты для эволюции базы данных.

---

## Что делает команда

Эта команда:

- Читает текущую схему
- Генерирует новую локальную миграцию
- Подготавливает артефакты, используемые Dinoco для эволюции базы данных

При желании она также может немедленно применить миграцию и сгенерировать модели Rust.

## Параметры

### --apply

Немедленно применяет сгенерированную миграцию, а также генерирует модели Rust.

Пример:

```bash
dinoco migrate generate --apply
```

## Пример использования без применения

```bash
dinoco migrate generate
```

Этот подход полезен, когда вы хотите:

- Проверить миграцию перед применением
- Просмотреть изменения в системе контроля версий
- Разделить генерацию и выполнение на разные этапы

## Пример использования с немедленным применением

```bash
dinoco migrate generate --apply
```

Этот подход полезен, когда вы хотите:

- Быстро обновить локальную базу данных
- Сгенерировать модели сразу после миграции
- Быстрее итерировать во время разработки

## Следующие шаги

После генерации вы можете:

```bash
dinoco migrate run
```

или:

```bash
dinoco models generate
```
