# Как использовать?

Команда `dinoco database reset` сбрасывает настроенную базу данных.

Это полезно при разработке, когда вы хотите полностью очистить текущее состояние базы данных.

---

## Что делает команда

При выполнении этой команды CLI пытается удалить объекты из текущей базы данных и воссоздать чистое состояние для настроенной среды.

Этот процесс особенно полезен для:

- Перезапуск локальной среды с нуля
- Очистка тестовых данных
- Исправление несогласованных сред во время разработки

## Меры предосторожности

- Эта команда разрушительна для настроенной базы данных.
- Используйте с осторожностью вне локальной среды.
- Всегда убеждайтесь, что URL-адрес подключения указывает на правильную базу данных.

## Следующие шаги

После сброса обычно выполняются следующие действия:

```bash
dinoco migrate run
```

или:

```bash
dinoco migrate generate --apply
```
