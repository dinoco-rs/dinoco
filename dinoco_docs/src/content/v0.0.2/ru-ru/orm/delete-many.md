# delete_many

Используется для удаления нескольких записей с одним или несколькими условиями.

---

## Что вы можете сделать

- `.cond(...)`: добавляет фильтры для пакетного удаления.
- `.execute(&client)`: выполняет удаление соответствующих записей.

## Возвращаемое значение

Возвращаемое значение `delete_many`:

```rust
DinocoResult<()>
```

## Базовый пример

Это удаляет только те записи, которые соответствуют условию.

```rust
dinoco::delete_many::<Session>()
    .cond(|w| w.expiresAt.lt(dinoco::Utc::now()))
    .execute(&client)
    .await?;
```

## Пример без фильтра

Это удаляет все данные из таблицы.

```rust
dinoco::delete_many::<TaskAssignees>()
    .execute(&client)
    .await?;
```
