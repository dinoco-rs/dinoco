export const title = "insert_into";

# insert_into

レコードを挿入するために使用されます。

---

## できること

- `.values(item)` で値を渡す
- `.with_relation(related)` でリレーションシップと共に挿入する
- `.with_connection(connected)` で既存のリレーションシップを接続する
- `.execute(&client)` で実行する

## メソッドの説明

- `.values(item)`: 挿入するレコードを定義します。
- `.with_relation(related)`: 親レコードと共に新しい関連レコードを挿入します。
- `.with_connection(connected)`: 挿入されたレコードを既存の要素に接続します。通常、接続をサポートするリレーションシップフローで使用されます。
- `.returning::<T>()`: 戻り値を、挿入された要素の型付きプロジェクションに変更します。
- `.execute(&client)`: データベースへの書き込み操作を実行します。

## 戻り値

`.returning::<T>()` なしの場合、戻り値は次のようになります。

```rust
DinocoResult<()>
```

`.returning::<T>()` を使用する場合、戻り値は次のようになります。

```rust
DinocoResult<T>
```

## 基本例

```rust
dinoco::insert_into::<User>()
    .values(User {
        id: "usr_1".to_string(),
        email: "ana@acme.com".to_string(),
        name: "Ana".to_string(),
    })
    .execute(&client)
    .await?;
```

## リレーション付きの例

生成されたモデルが親要素と関連要素の同時挿入をサポートしている場合は、`.with_relation(...)` を使用します。

```rust
dinoco::insert_into::<User>()
    .values(user)
    .with_relation(profile)
    .execute(&client)
    .await?;
```

## 接続付きの例

要素を挿入し、既存のリレーションシップを接続したい場合は、`.with_connection(...)` を使用します。

```rust
dinoco::insert_into::<User>()
    .values(new_user)
    .with_connection(existing_team)
    .execute(&client)
    .await?;
```

## 型付き戻り値の例

```rust
#[derive(Debug, Clone, dinoco::Extend)]
#[extend(User)]
struct UserSummary {
    id: i64,
    name: String,
}

let created = dinoco::insert_into::<User>()
    .values(User { id: 1, name: "Matheus".to_string() })
    .returning::<UserSummary>()
    .execute(&client)
    .await?;
```
