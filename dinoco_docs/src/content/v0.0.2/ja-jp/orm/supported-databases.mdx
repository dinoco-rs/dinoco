# 概要

**Dinoco** は、**PostgreSQL**、**MySQL**、**SQLite** のデータベースをネイティブでサポートしています。
近いうちに、このサポートを新しいデータベースに拡大する予定です！

> ⚠️
> 現在、Dinoco は**直接接続URL**のみをサポートしています。

---

## サポートされているバージョン

| データベース | 最小バージョン |
| :------------- | :------------ |
| **SQLite**     | `>= 3.25`     |
| **PostgreSQL** | `>= 10`       |
| **MySQL**      | `>= 8.0`      |

---

## 接続設定

設定は、スキーマファイルの `config` ブロック内で行います。

### SQLite

SQLiteを使用する場合、データベースファイルは `dinoco` フォルダ内に自動的に生成されます。

```dinoco
config {
  database = "sqlite"
  database_url = "file:./database.sqlite" # env("DATABASE_URL")
}
```

## PostgreSQL または MySQL

外部データベースの場合、対応する接続文字列を入力してください。

```dinoco
config {
  database = "postgresql" # または "mysql"
  database_url = env("DATABASE_URL")
}
```

## 型マッピング

| Dinoco       | Sqlite   | PostgreSQL       | MySQL            |
| :----------- | :------- | :--------------- | :--------------- |
| **String**   | TEXT     | TEXT             | VARCHAR(255)     |
| **Boolean**  | BOOLEAN  | BOOLEAN          | TINYINT(1)       |
| **Integer**  | INTEGER  | BIGINT           | BIGINT           |
| **Float**    | REAL     | DOUBLE PRECISION | DOUBLE PRECISION |
| **DateTime** | DATETIME | TIMESTAMP        | TIMESTAMP        |
| **Date**     | DATE     | DATE             | DATE             |
| **Json**     | BLOB     | JSONB            | JSON             |
