# Dinocoの紹介

このライブラリは、データベース管理を容易にすることを目的として作成されました。[Prisma](https://www.prisma.io/)の哲学に触発され、Dinocoは[Rust](https://www.rust-lang.org/)エコシステムに、流動的で型付けされた、非常に生産的なインターフェースをもたらします。

---

## RustにおけるPrisma体験

Dinocoは、Prismaの俊敏性と強力な型付けを[Rust](https://www.rust-lang.org/)エコシステムにもたらしたいという願望から生まれました。私たちは、日常の一般的な操作のために、複雑なマクロを設定したり、手動でSQLを書いたりするのに何時間も費やすべきではないと考えています。

## プロジェクトの柱

- **DX（開発者体験）を最優先:** 認知負荷を軽減し、開発を加速する流動的で直感的なAPI。
- **エンドツーエンドの型安全性:** データが常に型定義と同期していることを保証し、コンパイル時にエラーを捕捉します。
- **シンプルなリレーションシップ:** 面倒な手動結合は忘れてください。リレーションシップを定義すれば、Dinocoが舞台裏の複雑さを解決します。
- **強力なCLI:** ワークフローを整理し、迅速にするための統合されたマイグレーション管理とコード生成。

## 私たちの立ち位置

[Rust](https://www.rust-lang.org/)エコシステムには強力なツールがありますが、多くの場合、急な学習曲線が必要です。**Dinoco**は、現在の選択肢の間のギャップを埋めるために登場しました。

- **[Diesel](https://diesel.rs/)との比較:** DieselがSQLの深い知識と複雑なDSL（ドメイン固有言語）を要求するのに対し、Dinocoは、シンプルなクエリのためにコンパイラと格闘する必要なく、クリーンで生産性に焦点を当てた構文を優先します。
- **[SeaORM](https://www.sea-ql.org/SeaORM/)との比較:** SeaORMは優れていますが、そのレイヤーベースのアーキテクチャは多くのプロジェクトにとって過度に冗長になる可能性があります。Dinocoは不要な「定型コード」を排除し、はるかに少ないコード行で同じ堅牢性を提供します。

PrismaがNode.jsの世界にもたらした流動性とともに、[Rust](https://www.rust-lang.org/)のパフォーマンスを求めるなら、Dinocoはあなたの次のプロジェクトに最適な選択です。
