# 使用方法

`dinoco schema restore` コマンドは、ローカルのマイグレーションから `schema.dinoco` ファイルを復元します。

保存された履歴からスキーマを再構築する必要がある場合に便利です。

---

## コマンドの機能

このコマンドは以下を行います。

- ローカルのマイグレーションを読み取ります
- そのマイグレーションから保存されたメタデータを取得します
- `schema.dinoco` ファイルを再作成します

## 特定のマイグレーションからの復元例

```bash
dinoco schema restore 20250410_create_users
```

## 使用するタイミング

このコマンドは次の場合に使用します。

- `schema.dinoco` ファイルが失われた場合
- 履歴からローカルスキーマを再構築したい場合
- 特定のマイグレーションに保存されたスキーマを比較する必要がある場合

## 次のステップ

復元後、スキーマを確認し、新しいマイグレーションを生成または適用できます。
