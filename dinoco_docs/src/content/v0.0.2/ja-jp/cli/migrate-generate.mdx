# 使用方法

コマンド `dinoco migrate generate` は、現在のスキーマからマイグレーションを生成します。

これは、現在のスキーマの状態を既知の履歴と比較し、データベースを進化させるために必要なアーティファクトを作成します。

---

## コマンドの機能

このコマンドは以下を行います。

- 現在のスキーマを読み込みます
- 新しいローカルマイグレーションを生成します
- Dinocoがデータベースの進化に使用するアーティファクトを準備します

オプションで、マイグレーションをすぐに適用し、Rustモデルを生成することもできます。

## パラメータ

### --apply

生成されたマイグレーションをすぐに適用し、Rustモデルも生成します。

例:

```bash
dinoco migrate generate --apply
```

## 適用しない使用例

```bash
dinoco migrate generate
```

このフローは、次の場合に役立ちます。

- 適用前にマイグレーションを検査する
- バージョン管理の変更を確認する
- 生成と実行を異なるステップに分離する

## 即時適用する使用例

```bash
dinoco migrate generate --apply
```

このフローは、次の場合に役立ちます。

- ローカルデータベースを迅速に更新する
- マイグレーション直後にモデルを生成する
- 開発中に迅速に反復する

## 次のステップ

生成後、以下を実行できます。

```bash
dinoco migrate run
```

または：

```bash
dinoco models generate
```
