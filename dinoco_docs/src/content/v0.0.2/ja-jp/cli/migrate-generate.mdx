# 使い方

`dinoco migrate generate` コマンドは、現在の schema から migration を生成します。

現在の schema の状態を既知の履歴と比較し、データベースを進化させるために必要な artefact を生成します。

---

## コマンドの動作

このコマンドは:

- 現在のスキーマを読み取ります
- 新しいローカルマイグレーションを生成します
- Dinoco がデータベースの進化に使用するアーティファクトを準備します

オプションで、マイグレーションをすぐに適用し、Rust モデルを生成することもできます。

## パラメータ

### --apply

生成されたマイグレーションをすぐに適用し、Rust モデルも生成します。

Beispiel:

```bash
dinoco migrate generate --apply
```

## 適用せずに使う例

```bash
dinoco migrate generate
```

このワークフローは、次の場合に便利です:

- 適用前にマイグレーションを確認したい場合
- バージョン管理の変更を確認したい場合
- 生成と実行を異なるステップに分けたい場合

## すぐに適用する例

```bash
dinoco migrate generate --apply
```

このワークフローは、次の場合に便利です:

- ローカルデータベースをすばやく更新する
- マイグレーション後すぐにモデルを生成したい場合
- 開発プロセスでより迅速に反復したい場合

## 次のステップ

生成後、次のことができます:

```bash
dinoco migrate run
```

または:

```bash
dinoco models generate
```
