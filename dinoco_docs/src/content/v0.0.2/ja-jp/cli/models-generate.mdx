# 使い方

`dinoco models generate` コマンドは、データベースに保存された最新の migration から Rust モデルを生成します。

これは、生成されたコードを、すでに適用されているスキーマ構造と一致させるために使用されます。

---

## コマンドの動作

このコマンドは：

- 最新の利用可能なマイグレーションを読み取ります
- 生成されたRustモデルを再構築します
- 型付けされたDinoco APIによって使用されるアーティファクトを更新します

## 使うタイミング

次のようなときにこのコマンドを使います:

- すでにマイグレーションを適用しており、コードを再生成したい場合
- 構造的な変更後にモデルを更新する必要がある場合
- 型付けされたクライアントが最新の状態を反映していることを確認したい場合

## 次のステップ

生成後、モデルをRustコードで使用できるようになります。
