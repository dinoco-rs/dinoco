# 如何使用？

`dinoco schema restore` 命令从本地迁移中恢复 `schema.dinoco` 文件。

当您需要从保存的历史记录中重建模式时，这很有用。

---

## 此命令的作用

此命令会：

- 读取本地迁移
- 恢复该迁移中保存的元数据
- 重新创建 `schema.dinoco` 文件

## 从特定迁移恢复的示例

```bash
dinoco schema restore 20250410_create_users
```

## 何时使用

在以下情况下使用此命令：

- `schema.dinoco` 文件丢失
- 您想从历史记录中重建本地模式
- 您需要比较特定迁移中保存的模式

## 后续步骤

恢复后，您可以审查模式，然后生成或应用新的迁移。
