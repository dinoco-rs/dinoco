# 如何使用？

`dinoco database reset` 命令会重置已配置的数据库。

在开发中，如果你想彻底清空数据库当前状态，这会很有用。

---

## 命令的作用

执行该命令时，CLI 会尝试删除当前数据库中的对象，并为已配置环境重新建立一个干净状态。

此流程对于以下情况特别有用：

- 从头开始重新启动本地环境
- 清理测试数据
- 修复开发过程中不一致的环境

## 注意事项

- 这个命令会对已配置数据库执行破坏性操作。
- 在本地环境之外使用时请务必小心。
- 请始终确认连接 URL 指向正确的数据库。

## 下一步

重置后，您通常会执行以下操作：

```bash
dinoco migrate run
```

或：

```bash

dinoco migrate generate --apply
```
