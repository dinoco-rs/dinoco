# 如何使用？

`dinoco init` 命令会在当前项目中初始化 Dinoco 环境。

它会准备开始使用 schema、migration 和模型生成所需的基础。

---

## 命令的作用

执行 `dinoco init` 时，CLI 会引导完成 Dinoco 环境的初始配置。

通常，此流程会准备以下内容：

- `dinoco` 目录的基础结构
- 初始 Schema 文件
- 数据库配置
- 后续 migration 的基础

## 何时使用

当以下情况发生时使用这个命令：

- 你在项目中第一次配置 Dinoco
- 你想创建本地环境的初始结构
- 你需要为生成 migration 和模型准备项目

## 下一步

`init` 之后最常见的流程是：

```bash
dinoco migrate generate
```

或者，如果你想立即生成并应用：

```bash
dinoco migrate generate --apply
```
