# 如何使用？

命令 `dinoco init` 在当前项目中初始化 Dinoco 环境。

它为开始使用 schema、迁移和模型生成准备了必要的基础。

---

## 命令作用

执行 `dinoco init` 时，CLI 会引导 Dinoco 环境的初始配置。

通常，此流程会准备：

- `dinoco` 目录的基础结构
- 初始 schema 文件
- 数据库配置
- 未来迁移的基础

## 何时使用

在以下情况使用此命令：

- 您是首次在项目中配置 Dinoco
- 您想创建本地环境的初始结构
- 您需要准备项目以生成迁移和模型

## 后续步骤

在 `init` 之后，最常见的流程是：

```bash
dinoco migrate generate
```

或者，如果您想立即生成并应用：

```bash
dinoco migrate generate --apply
```
