# Dinoco 简介

这个库是为了简化数据库管理而设计的。受 [Prisma](https://www.prisma.io/) 哲学启发，Dinoco 为 [Rust](https://www.rust-lang.org/) 生态带来了流畅、类型安全且高效的接口。

---

## Rust 中的 Prisma 体验

Dinoco 源于把 Prisma 的敏捷性和强类型带入 [Rust](https://www.rust-lang.org/) 生态的愿望。我们认为，你不应该为了日常操作而花几个小时配置复杂宏或手写 SQL。

## 项目支柱

- **DX（开发者体验）优先：** 流畅直观的 API，减少认知负担并加速开发。
- **端到端类型安全：** 我们确保您的数据始终与类型定义同步，并在编译时捕获错误。
- **简单的关系：** 告别繁琐的手动连接。定义您的关系，让 Dinoco 在后台处理复杂性。
- **强大的 CLI：** 集成的迁移管理和代码生成，让您的工作流程保持有序和快速。

## 我们处于什么位置？

[Rust](https://www.rust-lang.org/) 生态系统拥有强大的工具，但通常需要陡峭的学习曲线。**Dinoco** 填补了现有选项之间的空白：

- **与 [Diesel](https://diesel.rs/) 相比：** 虽然 Diesel 需要深厚的 SQL 知识和复杂的 DSL（领域特定语言），但 Dinoco 优先提供简洁、注重生产力的语法，让您无需为简单的查询与编译器作斗争。
- **与 [SeaORM](https://www.sea-ql.org/SeaORM/) 相比：** SeaORM 非常出色，但其基于分层的架构对于许多项目来说可能过于冗余。Dinoco 消除了不必要的“样板”代码，并以更少的代码行提供相同的健壮性。

如果您正在寻找 [Rust](https://www.rust-lang.org/) 的强大功能与 Prisma 带给 Node.js 世界的流畅性，那么 Dinoco 是您下一个项目的正确选择。
