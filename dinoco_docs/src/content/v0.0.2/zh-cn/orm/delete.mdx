# delete

用于通过显式过滤器删除。

---

## 您可以做什么

- `.cond(...)`: 定义将要删除的记录。
- `.execute(&client)`: 在数据库中执行删除操作。

## 返回

`delete` 的返回值为：

```rust
DinocoResult<()>
```

## 基本示例

```rust
dinoco::delete::<User>()
    .cond(|w| w.id.eq(10))
    .execute(&client)
    .await?;
```

## 带有其他过滤器的示例

```rust
dinoco::delete::<Session>()
    .cond(|w| w.token.eq("session-1"))
    .execute(&client)
    .await?;
```
