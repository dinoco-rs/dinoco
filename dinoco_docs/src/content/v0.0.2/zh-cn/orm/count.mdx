export const title = "count";

# count

用于计数记录。

---

## 您可以做什么

- 使用 `.cond(...)` 过滤
- 使用 `.execute(&client)` 执行

## 方法描述

- `.cond(...)`: 限制哪些记录包含在计数中。
- `.execute(&client)`: 在数据库中执行计数。

## 返回

`count`的返回值为：

```rust
DinocoResult<usize>
```

## 基本示例

```rust
let total = dinoco::count::<User>()
    .execute(&client)
    .await?;
```

## 布尔过滤器示例

```rust
let total = dinoco::count::<User>()
    .cond(|w| w.active.eq(true))
    .execute(&client)
    .await?;
```

## 文本过滤器示例

```rust
let total = dinoco::count::<User>()
    .cond(|w| w.name.includes("Ana"))
    .execute(&client)
    .await?;
```
