export const title = "insert_into";

# insert_into

用于插入一条记录。

---

## 你可以做什么

- 通过 `.values(item)` 传递值
- 通过 `.with_relation(related)` 插入关联
- 通过 `.with_connection(connected)` 连接现有关联
- 通过 `.execute(&client)` 执行

## 方法说明

- `.values(item)`: 定义要插入的记录。
- `.with_relation(related)`: 与父级一起插入新的关联记录。
- `.with_connection(connected)`: 将插入的记录与现有元素连接，通常在支持连接的关系流中。
- `.returning::<T>()`: 将返回值更改为插入元素的类型化投影。
- `.execute(&client)`: 执行数据库写入操作。

## 返回值

如果没有 `.returning::<T>()`，则返回值为:

```rust
DinocoResult<()>
```

如果使用 `.returning::<T>()`，则返回值为:

```rust
DinocoResult<T>
```

## 简单示例

```rust
dinoco::insert_into::<User>()
    .values(User {
        id: "usr_1".to_string(),
        email: "ana@acme.com".to_string(),
        name: "Ana".to_string(),
    })
    .execute(&client)
    .await?;
```

## 关系示例

当生成的模型支持父级和关联项的联合插入时，请使用 `.with_relation(...)`。

```rust
dinoco::insert_into::<User>()
    .values(user)
    .with_relation(profile)
    .execute(&client)
    .await?;
```

## 连接示例

当您想要插入一个元素并连接一个已存在的关联时，请使用 `.with_connection(...)`。

```rust
dinoco::insert_into::<User>()
    .values(new_user)
    .with_connection(existing_team)
    .execute(&client)
    .await?;
```

## 带类型返回值的示例

```rust
#[derive(Debug, Clone, dinoco::Extend)]
#[extend(User)]
struct UserSummary {
    id: i64,
    name: String,
}

let created = dinoco::insert_into::<User>()
    .values(User { id: 1, name: "Matheus".to_string() })
    .returning::<UserSummary>()
    .execute(&client)
    .await?;
```
