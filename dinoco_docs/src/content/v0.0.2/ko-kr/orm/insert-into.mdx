export const title = "insert_into";

# insert_into

레코드를 삽입하는 데 사용됩니다.

---

## 할 수 있는 것

- `.values(item)`을 사용하여 값 전달
- `.with_relation(related)`을 통해 관계와 함께 삽입
- `.with_connection(connected)`을 통해 기존 관계 연결
- `.execute(&client)`로 실행

## 메서드 설명

- `.values(item)`: 삽입할 레코드를 정의합니다.
- `.with_relation(related)`: 부모 요소와 함께 새로운 관련 레코드를 삽입합니다.
- `.with_connection(connected)`: 삽입된 레코드를 이미 존재하는 요소에 연결합니다. 일반적으로 연결이 지원되는 관계 유형에서 사용됩니다.
- `.returning::<T>()`: 반환값을 삽입된 요소의 타입화된 프로젝션으로 변경합니다.
- `.execute(&client)`: 데이터베이스에 쓰기 작업을 실행합니다.

## 반환값

`.returning::<T>()`가 없으면 반환값은 다음과 같습니다:

```rust
DinocoResult<()>
```

`.returning::<T>()`가 있으면 반환값은 다음과 같습니다:

```rust
DinocoResult<T>
```

## 기본 예제

```rust
dinoco::insert_into::<User>()
    .values(User {
        id: "usr_1".to_string(),
        email: "ana@acme.com".to_string(),
        name: "Ana".to_string(),
    })
    .execute(&client)
    .await?;
```

## 관계 예제

생성된 모델이 부모 요소와 관련 요소의 동시 삽입을 지원하는 경우 `.with_relation(...)`을 사용하세요.

```rust
dinoco::insert_into::<User>()
    .values(user)
    .with_relation(profile)
    .execute(&client)
    .await?;
```

## 연결 예제

요소를 삽입하고 이미 존재하는 관계를 연결하려는 경우 `.with_connection(...)`을 사용하세요.

```rust
dinoco::insert_into::<User>()
    .values(new_user)
    .with_connection(existing_team)
    .execute(&client)
    .await?;
```

## 타입이 지정된 반환값 예제

```rust
#[derive(Debug, Clone, dinoco::Extend)]
#[extend(User)]
struct UserSummary {
    id: i64,
    name: String,
}

let created = dinoco::insert_into::<User>()
    .values(User { id: 1, name: "Matheus".to_string() })
    .returning::<UserSummary>()
    .execute(&client)
    .await?;
```
