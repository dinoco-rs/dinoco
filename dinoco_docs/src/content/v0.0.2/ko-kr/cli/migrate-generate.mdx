# 사용 방법?

`dinoco migrate generate` 명령어는 현재 스키마에서 마이그레이션을 생성합니다.

이 명령어는 현재 스키마 상태를 알려진 기록과 비교하고 데이터베이스를 발전시키는 데 필요한 아티팩트를 생성합니다.

---

## 명령어가 하는 일

이 명령어는 다음을 수행합니다:

- 현재 스키마를 읽습니다.
- 새로운 로컬 마이그레이션을 생성합니다.
- 데이터베이스 발전을 위해 Dinoco에서 사용하는 아티팩트를 준비합니다.

선택적으로, 마이그레이션을 즉시 적용하고 Rust 모델을 생성할 수도 있습니다.

## 매개변수

### --apply

생성된 마이그레이션을 즉시 적용하고 Rust 모델도 생성합니다.

예시:

```bash
dinoco migrate generate --apply
```

## 적용하지 않고 사용하는 예시

```bash
dinoco migrate generate
```

이 흐름은 다음을 원할 때 유용합니다:

- 적용하기 전에 마이그레이션을 검사합니다.
- 버전 제어의 변경 사항을 검토합니다.
- 생성과 실행을 다른 단계로 분리합니다.

## 즉시 적용하여 사용하는 예시

```bash
dinoco migrate generate --apply
```

이 흐름은 다음을 원할 때 유용합니다:

- 로컬 데이터베이스를 빠르게 업데이트합니다.
- 마이그레이션 직후 모델을 생성합니다.
- 개발 중에 더 빠르게 반복합니다.

## 다음 단계

생성 후 다음을 수행할 수 있습니다:

```bash
dinoco migrate run
```

또는:

```bash
dinoco models generate
```
