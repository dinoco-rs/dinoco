# 사용 방법

`dinoco migrate run` 명령은 모든 보류 중인 마이그레이션을 실행합니다.

이 명령은 구성된 데이터베이스를 이미 생성된 마이그레이션 기록과 동기화하는 데 사용됩니다.

---

## 명령이 하는 일

이 명령을 실행하면 CLI는 다음을 수행합니다:

- 마이그레이션 기록을 읽습니다.
- 아직 적용되지 않은 것을 확인합니다.
- 보류 중인 마이그레이션을 순서대로 실행합니다.

## 사용 시점

다음과 같은 경우 이 명령을 사용하십시오:

- 이미 마이그레이션을 생성한 경우
- 데이터베이스에 보류 중인 변경 사항을 적용하려는 경우
- 로컬 환경 또는 배포 환경을 준비하는 경우

## 다음 단계

필요한 경우 다음을 사용하여 모델을 생성하십시오:

```bash
dinoco models generate
```
