# 사용 방법

`dinoco init` 명령은 현재 프로젝트에서 Dinoco 환경을 초기화합니다.

이 명령은 스키마, 마이그레이션 및 모델 생성을 시작하는 데 필요한 기반을 준비합니다.

---

## 명령이 하는 일

`dinoco init`을 실행하면 CLI가 Dinoco 환경의 초기 구성을 안내합니다.

일반적으로 이 흐름은 다음을 준비합니다:

- `dinoco` 디렉토리의 기본 구조
- 초기 스키마 파일
- 데이터베이스 구성
- 향후 마이그레이션을 위한 기반

## 사용 시점

다음 경우에 이 명령을 사용하십시오:

- 프로젝트에 Dinoco를 처음 설정할 때
- 로컬 환경의 초기 구조를 생성하고자 할 때
- 마이그레이션 및 모델 생성을 위해 프로젝트를 준비해야 할 때

## 다음 단계

`init` 후 가장 일반적인 흐름은 다음과 같습니다:

```bash
dinoco migrate generate
```

또는 즉시 생성하고 적용하려면:

```bash
dinoco migrate generate --apply
```
