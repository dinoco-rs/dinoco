# 사용 방법?

`dinoco database reset` 명령어는 구성된 데이터베이스를 재설정합니다.

개발 시 현재 데이터베이스 상태를 완전히 지우고 싶을 때 유용합니다.

---

## 명령어가 하는 일

이 명령어를 실행하면 CLI는 현재 데이터베이스에서 객체를 제거하고 구성된 환경에 대해 깨끗한 상태를 다시 생성하려고 시도합니다.

이 흐름은 특히 다음 경우에 유용합니다:

- 로컬 환경을 처음부터 다시 시작할 때
- 테스트 데이터 정리
- 개발 중 일관성 없는 환경 수정

## 주의사항

- 이 명령어는 구성된 데이터베이스에 파괴적입니다.
- 로컬 환경 외부에서는 주의해서 사용하십시오.
- 항상 연결 URL이 올바른 데이터베이스를 가리키는지 확인하십시오.

## 다음 단계

재설정 후에는 일반적으로 다음을 실행합니다:

```bash
dinoco migrate run
```

또는:

```bash
dinoco migrate generate --apply
```
