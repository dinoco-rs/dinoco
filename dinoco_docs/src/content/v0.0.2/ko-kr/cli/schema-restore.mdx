# 사용 방법

명령어 `dinoco schema restore`는 파일 `schema.dinoco`을(를) 로컬 마이그레이션에서 복원합니다.

이것은 저장된 기록에서 스키마를 재구성해야 할 때 유용합니다.

---

## 명령이 하는 일

이 명령어는:

- 로컬 마이그레이션을 읽습니다.
- 해당 마이그레이션의 저장된 메타데이터를 검색합니다.
- `schema.dinoco` 파일을 다시 생성합니다.

## 특정 마이그레이션에서 복원하는 예제

```bash
dinoco schema restore 20250410_create_users
```

## 사용 시점

다음과 같은 경우 이 명령어를 사용하십시오:

- `schema.dinoco` 파일이 손실되었을 때
- 기록에서 로컬 스키마를 재구성하려는 경우
- 특정 마이그레이션에서 저장된 스키마를 비교해야 할 때

## 다음 단계

복원 후, 스키마를 확인하고 새 마이그레이션을 생성하거나 적용할 수 있습니다.
