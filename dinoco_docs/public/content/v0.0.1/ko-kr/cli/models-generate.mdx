# 사용 방법?

`dinoco models generate` 명령어는 데이터베이스에 저장된 최신 마이그레이션으로부터 Rust 모델을 생성합니다.

이는 이미 적용된 스키마 구조와 생성된 코드를 일치시키기 위해 사용됩니다.

---

## 명령어가 하는 일

이 명령어는 다음을 수행합니다:

- 사용 가능한 최신 마이그레이션을 읽습니다.
- 생성된 Rust 모델을 재구축합니다.
- Dinoco의 타입 지정 API에서 사용되는 아티팩트를 업데이트합니다.

## 언제 사용해야 하나요?

다음과 같은 경우 이 명령어를 사용하세요:

- 이미 마이그레이션을 적용했고 코드를 다시 생성하고 싶을 때
- 구조적 변경 후 모델을 업데이트해야 할 때
- 타입 지정 클라이언트가 최신 상태를 반영하도록 보장하고 싶을 때
