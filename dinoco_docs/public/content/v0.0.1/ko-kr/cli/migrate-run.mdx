# 사용 방법?

`dinoco migrate run` 명령어는 모든 보류 중인 마이그레이션을 실행합니다.

이 명령어는 구성된 데이터베이스를 이미 생성된 마이그레이션 기록과 일치시키는 데 사용됩니다.

---

## 명령어가 하는 일

이 명령어를 실행하면 CLI는 다음을 수행합니다:

- 마이그레이션 기록을 읽습니다.
- 아직 적용되지 않은 마이그레이션을 확인합니다.
- 보류 중인 마이그레이션을 순서대로 실행합니다.

## 사용 시기

다음과 같은 경우에 이 명령어를 사용하세요:

- 이미 생성된 마이그레이션이 있는 경우
- 데이터베이스에 보류 중인 변경 사항을 적용하려는 경우
- 로컬 또는 배포 환경을 준비 중인 경우

## 다음 단계

필요한 경우 다음 명령어로 모델을 생성하세요:

dinoco models generate
