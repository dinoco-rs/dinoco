# 사용 방법?

`dinoco schema restore` 명령어는 로컬 마이그레이션에서 `schema.dinoco` 파일을 복원합니다.

저장된 기록에서 스키마를 재구축해야 할 때 유용합니다.

---

## 명령어가 하는 일

이 명령어는 다음을 수행합니다:

- 로컬 마이그레이션을 읽습니다.
- 해당 마이그레이션에 저장된 메타데이터를 검색합니다.
- `schema.dinoco` 파일을 다시 생성합니다.

## 특정 마이그레이션에서 복원하는 예시

```bash
dinoco schema restore 20250410_create_users
```

## 사용 시점

다음 경우에 이 명령어를 사용하세요:

- `schema.dinoco` 파일이 손실되었을 때
- 기록에서 로컬 스키마를 재구축하고 싶을 때
- 특정 마이그레이션에 저장된 스키마를 비교해야 할 때

## 다음 단계

복원 후, 스키마를 검토하고 새로운 마이그레이션을 생성하거나 적용할 수 있습니다.
