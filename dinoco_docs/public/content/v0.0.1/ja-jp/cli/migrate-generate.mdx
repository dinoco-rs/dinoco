# 使用方法

コマンド `dinoco migrate generate` は、現在のスキーマからマイグレーションを生成します。

これは、現在のスキーマの状態を既知の履歴と比較し、データベースを進化させるために必要なアーティファクトを作成します。

---

## このコマンドの機能

このコマンドは以下のことを行います:

- 現在のスキーマを読み取ります
- 新しいローカルマイグレーションを生成します
- Dinoco がデータベースの進化に使用するアーティファクトを準備します

オプションで、マイグレーションをすぐに適用し、Rust モデルを生成することもできます。

## パラメータ

### --apply

生成されたマイグレーションをすぐに適用し、Rust モデルも生成します。

例:

```bash
dinoco migrate generate --apply
```

## 適用せずに使用する例

```bash
dinoco migrate generate
```

このフローは、以下の場合に役立ちます:

- 適用前にマイグレーションを検査する
- バージョン管理の変更を確認する
- 生成と実行を異なるステップに分ける

## すぐに適用して使用する例

```bash
dinoco migrate generate --apply
```

このフローは、以下の場合に役立ちます:

- ローカルデータベースを迅速に更新する
- マイグレーション直後にモデルを生成する
- 開発中に迅速に反復する

## 次のステップ

生成後、以下を実行できます:

```bash
dinoco migrate run
```

または:

```bash
dinoco models generate
```
