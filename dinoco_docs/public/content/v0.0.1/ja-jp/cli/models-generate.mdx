# 使い方？

`dinoco models generate` コマンドは、データベースに保存されている最新のマイグレーションからRustモデルを生成します。

これは、生成されたコードが既に適用されているスキーマ構造と一致するようにするために使用されます。

---

## このコマンドの機能

このコマンドは以下を実行します:

- 利用可能な最新のマイグレーションを読み取ります
- 生成されたRustモデルを再構築します
- Dinocoの型付きAPIで使用されるアーティファクトを更新します

## 使用するタイミング

このコマンドは以下の状況で使用します:

- すでにマイグレーションを適用しており、コードを再生成したい場合
- 構造的な変更後にモデルを更新する必要がある場合
- 型付きクライアントが最新の状態を反映していることを確認したい場合

## 次のステップ

生成後、Rustコードでモデルを使用できるようになります。
