export const title = "insert_into";

# insert_into

レコードを挿入するために使用されます。

---

## できること

- `.values(item)` で値を渡す
- `.with_relation(related)` で関連を付けて挿入する
- `.with_connection(connected)` で既存の関連を接続する
- `.execute(&client)` で実行する

## メソッドの説明

- `.values(item)`: 挿入するレコードを定義します。
- `.with_relation(related)`: 親と一緒に新しい関連レコードを挿入します。
- `.with_connection(connected)`: 挿入されたレコードを既存のアイテムに接続します。通常、接続がサポートされている関連フローで使用されます。
- `.returning::&lt;T&gt;()`: 戻り値を、挿入されたアイテムの型付きプロジェクションに変更します。
- `.execute(&client)`: データベースへの書き込みを実行します。

## 戻り値

`.returning::&lt;T&gt;()` なしの場合、戻り値は次のとおりです。

```rust
DinocoResult<()>
```

`.returning::&lt;T&gt;()` ありの場合、戻り値は次のようになります。

```rust
DinocoResult<T>
```

## シンプルな例

```rust
dinoco::insert_into::<User>()
    .values(User {
        id: "usr_1".to_string(),
        email: "ana@acme.com".to_string(),
        name: "Ana".to_string(),
    })
    .execute(&client)
    .await?;
```

## 関連を伴う例

生成されたモデルが親と関連を一緒に挿入することをサポートしている場合、`.with_relation(...)` を使用します。

```rust
dinoco::insert_into::<User>()
    .values(user)
    .with_relation(profile)
    .execute(&client)
    .await?;
```

## 接続を伴う例

アイテムを挿入し、既存の関連を接続したい場合、`.with_connection(...)` を使用します。

```rust
dinoco::insert_into::<User>()
    .values(new_user)
    .with_connection(existing_team)
    .execute(&client)
    .await?;
```

## 型付き戻り値を伴う例

```rust
#[derive(Debug, Clone, dinoco::Extend)]
#[extend(User)]
struct UserSummary {
    id: i64,
    name: String,
}

let created = dinoco::insert_into::<User>()
    .values(User { id: 1, name: "Matheus".to_string() })
    .returning::<UserSummary>()
    .execute(&client)
    .await?;
```
