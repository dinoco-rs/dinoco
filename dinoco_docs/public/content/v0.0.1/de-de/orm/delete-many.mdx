export const title = "delete_many";

# delete_many

Wird verwendet, um mehrere Datensätze mit einer oder mehreren Bedingungen zu löschen.

---

## Was Sie tun können

- `.cond(...)`: fügt Filter für die Stapellöschung hinzu.
- `.execute(&client)`: führt die Löschung der entsprechenden Datensätze aus.

## Rückgabe

Die Rückgabe von `delete_many` ist:

```rust
DinocoResult<()>
```

## Grundlegendes Beispiel

Dies löscht nur die Datensätze, die die Bedingung erfüllen.

```rust
dinoco::delete_many::<Session>()
    .cond(|w| w.expiresAt.lt(dinoco::Utc::now()))
    .execute(&client)
    .await?;
```

## Beispiel ohne Filter

Dies löscht alle Daten aus der Tabelle.

```rust
dinoco::delete_many::<TaskAssignees>()
    .execute(&client)
    .await?;
```
