# delete

用于通过显式过滤器删除。

---

## 您可以做什么

- `.cond(...)`: 定义将删除哪个记录。
- `.execute(&client)`: 在数据库中执行删除。

## 返回值

`delete` 的返回值是：

```rust
DinocoResult<()>
```

## 基本示例

```rust
dinoco::delete::<User>()
    .cond(|w| w.id.eq(10))
    .execute(&client)
    .await?;
```

## 带有其他过滤器的示例

```rust
dinoco::delete::<Session>()
    .cond(|w| w.token.eq("session-1"))
    .execute(&client)
    .await?;
```
