export const title = "数据库重置";

# 如何使用？

命令 `dinoco database reset` 重置配置的数据库。

在开发中，当您希望完全清除数据库的当前状态时，此命令非常有用。

---

## 命令的作用

执行此命令时，CLI 会尝试删除当前数据库中的对象，并为配置的环境重新创建一个干净的状态。

此流程特别适用于：

- 从头开始重新启动本地环境
- 清除测试数据
- 修复开发过程中不一致的环境

## 注意事项

- 此命令对配置的数据库具有破坏性。
- 在本地环境之外使用时请务必谨慎。
- 请务必确认连接 URL 指向正确的数据库。

## 后续步骤

重置后，您通常会执行：

```bash
dinoco migrate run
```

或：

```bash
dinoco migrate generate --apply
```
