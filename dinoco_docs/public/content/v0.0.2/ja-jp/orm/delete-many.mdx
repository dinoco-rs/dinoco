export const title = "delete_many";

# delete_many

1つ以上の条件で複数のレコードを削除するために使用されます。

---

## できること

- `.cond(...)`: バッチ削除のフィルターを追加します。
- `.execute(&client)`: 一致するレコードの削除を実行します。

## 戻り値

`delete_many`の戻り値は次のとおりです。

```rust
DinocoResult<()>
```

## 基本的な例

これは、条件を満たすレコードのみを削除します。

```rust
dinoco::delete_many::<Session>()
    .cond(|w| w.expiresAt.lt(dinoco::Utc::now()))
    .execute(&client)
    .await?;
```

## フィルターなしの例

これは、テーブル内のすべてのデータを削除します。

```rust
dinoco::delete_many::<TaskAssignees>()
    .execute(&client)
    .await?;
```
