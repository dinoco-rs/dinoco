# 사용 방법

`dinoco schema restore` 명령어는 로컬 마이그레이션에서 `schema.dinoco` 파일을 복원합니다.

저장된 기록에서 스키마를 재구성해야 할 때 유용합니다.

---

## 명령어가 하는 일

이 명령어는 다음을 수행합니다:

- 로컬 마이그레이션을 읽습니다.
- 해당 마이그레이션에 저장된 메타데이터를 검색합니다.
- `schema.dinoco` 파일을 다시 생성합니다.

## 특정 마이그레이션에서 복원하는 예시

```bash
dinoco schema restore 20250410_create_users
```

## 언제 사용해야 하나요

다음과 같은 경우 이 명령어를 사용하세요:

- `schema.dinoco` 파일이 손실된 경우
- 기록에서 로컬 스키마를 재구성하려는 경우
- 특정 마이그레이션에 저장된 스키마를 비교해야 하는 경우

## 다음 단계

복원 후 스키마를 검토한 다음 새 마이그레이션을 생성하거나 적용할 수 있습니다.
